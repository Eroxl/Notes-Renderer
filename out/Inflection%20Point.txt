<p>An inflection point is any point on a graph of a function where the <a href="Concavity">concavity</a> of the function changes (ie. from concave up to concave down). For a function to have an inflection point it must have a <a href="Higher%20Order%20Derivatives">second derivative</a>. Inflection points can be found on a graph by checking for locations where the <a href="Higher%20Order%20Derivatives">second derivative</a> switches signs.</p>
