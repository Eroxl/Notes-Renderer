<p>A differentiable function is any function where the <a href="Derivative">derivative</a> exists at every point in its <a href="Domain">domain</a>. This means that for every point on the graph of the function there exists a non-vertical <a href="Tangent%20Line">tangent line</a>. A differentiable function is also sometimes described as being &quot;smooth&quot; and does not contain any breaks or cusps. For a point to be differentiable it also needs to be <a href="Continuous%20Function">continous</a>.</p>
<p>Differentiability can also be described as any point where the following limit (the limit definition of the derivative) exists.</p>
