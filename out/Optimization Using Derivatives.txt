<p><a href="Derivative">Derivatives</a> can be used as a tool for maximizing or minimizing the output of a function.</p>
<h1>Steps</h1>
<ol>
<li>Draw an appropriate figure and label the quantities relevant to the problem. These quantities usually take the form of multiple variables.</li>
<li>Find a formula for the quantity to be maximized or minimized.</li>
<li>Express the formula that is being maximized or minimized as a function of a single variable. This step usually requires rearranging the equations in step #1.</li>
<li>Find the interval of possible values for this variable from the physical restrictions in the problem.</li>
<li>Use calculus techniques to obtain the absolute maximum or minimum.</li>
</ol>

<p>The first derivative test is a method for finding <a href="Local%20Extrema">local extrema</a> of a function at a <a href="Critical%20Point">critical point</a>.</p>
<p>The first derivative test is performed by checking the first <a href="Derivative">derivative</a> right before the point being checked and right after. If the <a href="Derivative">derivative</a> switches from positive to negative there is a <a href="Local%20Maximum">local maximum</a> there, alternatively if the <a href="Derivative">derivative</a> switches from negative to positive there is a <a href="Local%20Minimum">local minimum</a> there. If the sign of the <a href="Derivative">derivative</a> stays the same there is no <a href="Local%20Extrema">extrema</a> at that point.</p>

                Second Derivative Test
              </div>
              <div class="embed-content">
                <p><mjx-container class="MathJax" jax="CHTML" style="font-size: 121%;" display="true"><mjx-math display="true" style="margin-left: 0; margin-right: 0;" class=" MJX-TEX"></mjx-math></mjx-container></p>
<p>The second derivative test is a method for potentially finding <a href="Local%20Extrema">local extrema</a> of a function at a <a href="Critical%20Point">critical point</a>. The second derivative test can sometimes be inconclusive in which case the <a href="First%20Derivative%20Test">first derivative test</a> must be used.</p>
<p>The second derivative test is performed by checking the <a href="Higher%20Order%20Derivatives">second derivative</a> at a <a href="Critical%20Point">critical point</a>.</p>
<p>The test can result in 3 different cases</p>
<ul>
<li>If the <a href="Higher%20Order%20Derivatives">second derivative</a> is negative there is a <a href="Local%20Maximum">local maximum</a></li>
