<p>A call graph is a type of <a href="Graph">graph</a> that shows the relationships between different <a href="Method">methods</a>. Each <a href="Graph#Node">node</a> represents a <a href="Method">method</a> and each <a href="Graph#Edge">edge</a> (the lines on the <a href="Graph">graph</a>) represent that the <a href="Method">method</a> &quot;A&quot; <a href="Method#Call">calls</a> <a href="Method">method</a> &quot;B&quot;.</p>
<h1>Example</h1>
<p>Consider the following code</p>
<pre><code><span class="hljs-keyword">class</span> <span class="hljs-title class_">JDayChooser</span> {

   Calendar calendar;
   
   JDayChooser() {
      calendar = <span class="hljs-keyword">new</span> <span class="hljs-title class_">Calendar</span>();

   }

   <span class="hljs-keyword">void</span> <span class="hljs-title function_">init</span><span class="hljs-params">()</span> {
      drawDayNames();
      drawDays();
   }

   <span class="hljs-keyword">void</span> <span class="hljs-title function_">drawDayNames</span><span class="hljs-params">()</span> {
      <span class="hljs-type">int</span> <span class="hljs-variable">firstDayOfWeek</span> <span class="hljs-operator">=</span> calendar.getFirstDayOfWeek();
   }

   <span class="hljs-keyword">void</span> <span class="hljs-title function_">drawDays</span><span class="hljs-params">()</span> { }
}
</code></pre><p>As is shown in the code the <a href="Constructor">constructor</a> <code>JDayChooser</code> <a href="Method#Call">calls</a> both <code>Calendar</code> through the line <code>calendar = new Calendar();</code>, as well as <code>init</code> through the line <code>init();</code> so they are both connected to the <a href="Constructor">constructor</a>.</p>
<p>Because <code>Calendar</code> is not in the <a href="Package">package</a> were analyzing we omit the sub <a href="Method#Call">calls</a> of it so it has no arrows out of it. The next <a href="Method">methods</a> are called from <code>init</code>, which are the calls to <code>drawDayNames</code> and <code>drawDays</code> which both get arrows connecting them to <code>init</code>.</p>
<p>As is shown in the code <code>drawDays</code> doesn't <a href="Method#Call">call</a> any other <a href="Method">methods</a> and <code>drawDayNames</code> calls <code>Calendar.getFirstDayOfWeek</code> with the code <code>int firstDayOfWeek = calendar.getFirstDayOfWeek();</code> so it is connected to <code>drawDayNames</code>.</p>
